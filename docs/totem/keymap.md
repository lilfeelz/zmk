## Layer Scheme

| Layer | Trigger | Purpose |
|---|---|---|---|
| BASE (0) | default | Alpha keys |
| NAV (1) | hold left thumb inner | Numbers, navigation, editing (undo/copy/paste), app switching |
| SYM (2) | hold right thumb inner | Symbols, brackets, punctuation |
| MOO (3) | hold right thumb outer | F-keys, sticky modifiers |
| FUN (4) | hold either thumb outer | Bluetooth, display brightness, volume |
