# TOTEM ZMK Configuration

ZMK firmware configuration for the [TOTEM](https://github.com/GEIGEIGEIST/TOTEM) keyboard — a 38-key split column-staggered keyboard powered by Seeed XIAO BLE.
[![GitHub](https://img.shields.io/badge/GitHub-lilfeelz%2Fzmk-181717?style=flat&logo=github)](https://github.com/lilfeelz/zmk)

## Layer Scheme

| Layer | Trigger | Purpose |
|---|---|---|
| BASE (0) | default | Alpha keys |
| NAV (1) | hold left thumb | Navigation, media, F-keys, Bluetooth |
| SYM (2) | hold right thumb | Symbols, numbers, brightness |
| MOO (3) | hold both thumbs | Mouse keys |
